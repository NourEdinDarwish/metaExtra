# Changelog

## metaExtra (development version)

- New
  [`forest_save()`](https://nouredindarwish.github.io/metaExtra/reference/forest_save.md)
  saves a `meta` forest plot to disk with automatically calculated ideal
  dimensions from the forest plot’s internal grid layout.
- New
  [`forest_dims()`](https://nouredindarwish.github.io/metaExtra/reference/forest_dims.md)
  calculates the optimal width and height of a `meta` forest plot for
  saving.
